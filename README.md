# portchecktool-api

Unofficial https://www.portchecktool.com/ wrapper written in TypeScript

## Building

As a NodeJS module:

```bash
tsc --outDir ./dist/module/
```

and to bundle for browsers:

```bash
browserify -o ./dist/browserify/bundle.js ./dist/module/index.js
```
